Fig. 01 — Function map

One controller, the whole rig

NOTHING LEFT DARK
01

Clip launch grid

The 4×4 pad grid fires Launcher clips with live colour & state LED feedback.

02

Scene launch

Top row launches whole scenes; session arrows move the focus box, plus Stop-All.

03

Snapshot → scene

Fire any scene from a Drop snapshot via Program Change — target set per snapshot, live.

04

64-channel mixer

8 faders across 8 layers (A–H) reach channels 1–64, with matching mutes.

05

32 assignable knobs

Encoders map to free Knob controls; 32 pushes become momentary buttons via Control Link.

06

Tempo master

Studio One drives the Drop as clock master — set the Drop to Ext MIDI and it locks to tempo.

Two independent paths. In DAW mode the grid launches clips and scenes; in snapshot mode a Program Change fires a scene by number — with no viewport movement. They never cross, so you can arrange with one and recall with the other.

Four steps

macOS / WINDOWS
01

Prepare the Drop

Import drop-config/daw-init-studioone.json (back up your own first).

02

Install the device

Copy studio-one/Neuzeit/Drop/ into your Studio One User Devices folder.

03

Add it in Studio One

External Devices ▸ Add ▸ Neuzeit ▸ Drop. Set Receive-From & Send-To to the Drop's port.

04

Play

Open the Launcher — the grid lights up and launches clips.
